# C-Statement-Expression

此專案示範 C 的陳述與運算式。

在終端機編譯執行 demo1.c ，結果如下

```
$ gcc demo1.c
$ a.out
請輸入第一個數字：2
請輸入第二個數字：3
請輸入計算符號 + - * /：+
結果是 5 。
$
```

在終端機編譯執行 demo2.c ，結果如下

```
$ gcc demo2.c
$ a.out
1 - 向右位移 1 位元。
2 - 向右位移 2 位元。
3 - 向左位移 4 位元。
4 - 向左位移 8 位元。
目前 n 為 8 ，請輸入你的選項：
3
n 的結果是 128 。
$
```
